The `Event` interface defines the structure of an event object that the calendar components use to display and manage events.

## Definition

```ts
export interface Event {
  id: string;
  title: string;
  start: string;
  end: string;
  color: EventColor;
}
```

## Example: One Event
```ts
const event: Event =
  {
    id: "1",
    title: "Morning Meeting",
    start: "2025-03-28T09:00:00",
    end: "2025-03-28T10:00:00",
    color: {
        background: "#10b981", 
        foreground: "#ffffff",
      },
  }
```

## Example : Multiple Events

```ts
const events: Event[] = [
  {
    id: "1",
    title: "Morning Meeting",
    start: "2025-03-28T09:00:00",

        background: "#10b981", 
        foreground: "#ffffff",
      },
  },
  {
    id: "2",
    title: "Coffee Chat",
    start: "2025-03-28T09:00:00",
    end: "2025-03-28T09:15:00",

      foreground: "#ffffff",
    },
  }

```
