# Quickstart

Welcome to the quickstart guide for **Solisa Calendar**. This guide will walk you through the setup and basic integration of our calendar components into your React project.

## Prerequisites

Before you begin, ensure that you have:
- A working React environment (e.g., using [Next.js](https://nextjs.org/) or Create React App).
- Node.js installed on your machine.

## Installation

Install the `solisa-calendar` package using your preferred package manager. For example:

```bash
npm install solisa-calendar
# or
yarn add solisa-calendar
```
